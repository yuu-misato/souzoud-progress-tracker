# SOUZOUD Inc. - Progress Tracker

制作現場向けプロジェクト進捗共有サービス

## 機能

- 🏢 クライアントIDで複数プロジェクトにアクセス
- 📋 カスタム工程の作成・編集・削除
- 📝 各工程に仕様・要件を記録
- ✨ 明るく爽やかなライトテーマUI

## 使い方

### クライアント向け

1. クライアントIDを入力
2. プロジェクト一覧から確認したいプロジェクトを選択
3. 進捗状況とタイムラインを確認

### 管理者向け

1. 管理画面にアクセス
2. プロジェクトの作成・編集
3. 工程のステータス更新と仕様入力

## ローカル開発

```bash
python3 -m http.server 8080
```
